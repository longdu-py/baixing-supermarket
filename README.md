# 百姓量贩连锁超市官网
本地连锁超市静态展示商城网站，纯前端HTML/CSS/JS，支持GitHub Pages一键部署。

## 功能介绍
1. 首页轮播广告、热销商品、优惠活动展示
2. 生鲜/零食/粮油/日化商品分类浏览
3. 本地购物车本地存储（刷新不丢失）
4. 濮阳线下门店查询筛选
5. 企业介绍、客服联系方式、留言表单

## 使用方法
1. 克隆仓库：git clone https://github.com/你的用户名/baixing-supermarket.git
2. 直接打开index.html本地预览
3. 仓库开启GitHub Pages即可线上访问

## 技术栈
HTML5 + Bootstrap5 + CSS3 + JavaScript(本地存储localStorage)
